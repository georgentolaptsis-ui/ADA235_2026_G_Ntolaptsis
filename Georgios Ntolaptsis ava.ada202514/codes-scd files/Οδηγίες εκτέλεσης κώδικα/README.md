## Περιγραφή  
  
Η παρούσα εργασία αποτελείται από τρία αρχεία SuperCollider (.scd), τα οποία παράγουν και συνδυάζουν ηχητικά αποτελέσματα.  
  
---  
  
## Αρχεία  
  
- `drum loop.scd`  
Αυτόνομο ηχητικό script (drum loop)  
  
- `sun o))).recon.scd`  
Αυτόνομο ηχητικό script που χρησιμοποιεί εξωτερικό αρχείο ήχου  
  
- `combined.scd`  
Script που συνδυάζει τα παραπάνω δύο  
  
---
