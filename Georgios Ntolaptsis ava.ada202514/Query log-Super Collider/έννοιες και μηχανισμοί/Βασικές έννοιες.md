
[[Τρόπος λειτουργείας Collider]]


1. 🔧 Συνάρτηση (Function)

Ένα κομμάτι κώδικα που **κάνει κάτι** και μπορείς να το καλείς όποτε θέλεις.

supercollider

```supercollider
// Ορισμός συνάρτησης
f = { |freq, amp|
    SinOsc.ar(freq, 0, amp)
};

// Κλήση συνάρτησης
f.value(440, 0.5);  // παράγει ήχο 440Hz
f.value(880, 0.3);  // παράγει ήχο 880Hz
```

Η συνάρτηση **δεν θυμάται** τίποτα — απλώς παίρνει είσοδο και δίνει έξοδο.

---

### 2. 🏛️ Κλάση (Class)

Ένα **"καλούπι"** που περιγράφει τι είναι κάτι και τι μπορεί να κάνει. Από μία κλάση φτιάχνεις **αντικείμενα**.

supercollider

```supercollider
// Η SinOsc είναι κλάση
// Δημιουργείς αντικείμενο από αυτή:
x = SinOsc.ar(440, 0, 0.5);

// Η Array είναι κλάση
a = Array.new(4);
a.add(1);
a.add(2);

// Η Buffer είναι κλάση
b = Buffer.alloc(s, 44100);
```

Μπορείς να φτιάξεις και δική σου κλάση (σε αρχείο `.sc`):

supercollider

```supercollider
MyOscillator {
    var freq, amp;

    *new { |freq, amp|
        ^super.new.init(freq, amp)
    }

    init { |f, a|
        freq = f;
        amp = a;
    }

    play {
        { SinOsc.ar(freq, 0, amp) }.play
    }
}

// Χρήση:
o = MyOscillator.new(440, 0.5);
o.play;
```

---

### 3. 📄 Script

Ένα **αρχείο κώδικα** (`.scd`) που εκτελείς γραμμή-γραμμή ή ολόκληρο. Δεν είναι κλάση, είναι απλώς οδηγίες.

supercollider

```supercollider
// myScript.scd

s.boot;

SynthDef(\sine, { |freq = 440, amp = 0.5|
    Out.ar(0, SinOsc.ar(freq, 0, amp) * EnvGen.kr(Env.perc, doneAction: 2))
}).add;

Synth(\sine, [freq: 660, amp: 0.4]);
```

Το script είναι σαν μια **συνταγή** — το διαβάζεις από πάνω προς τα κάτω.

---

### 4. 💾 State (Κατάσταση)

Οι **τιμές που "θυμάται"** κάτι σε μια δεδομένη στιγμή.

supercollider

```supercollider
// State σε μεταβλητή
~tempo = 120;
~scale = Scale.minor;
~volume = 0.5;

// Αλλαγή state
~tempo = 140;  // τώρα παίζει πιο γρήγορα

// State σε αντικείμενο
p = Pbind(
    \degree, Pseq([0, 2, 4], inf),
    \dur, 0.5
);
// Το p "θυμάται" τις ρυθμίσεις του
```

Το state μπορεί να **αλλάζει με τον χρόνο**:

supercollider

```supercollider
~synth = Synth(\sine, [freq: 440]);

// Αργότερα αλλάζεις το state του:
~synth.set(\freq, 880);
~synth.set(\amp, 0.2);
```

---

### 5. 🎭 Συμπεριφορά (Behavior)

Το **τι κάνει** ένα αντικείμενο — οι μέθοδοί του. Η συμπεριφορά συνδέεται με το state.

supercollider

```supercollider
// Το Synth έχει συμπεριφορές:
x = Synth(\sine);
x.run(false);   // σταματά
x.run(true);    // ξαναρχίζει
x.set(\freq, 660);  // αλλάζει παράμετρο
x.free;         // σβήνει

// Το Array έχει συμπεριφορές:
a = [1, 2, 3, 4, 5];
a.reverse;      // [5,4,3,2,1]
a.scramble;     // τυχαία σειρά
a.mirror;       // [1,2,3,4,5,4,3,2,1]
```

---

### 🗺️ Πώς συνδέονται μεταξύ τους

```
Κλάση  →  ορίζει τη ΔΟΜΗ και τη ΣΥΜΠΕΡΙΦΟΡΑ
   ↓
Αντικείμενο  →  έχει STATE (συγκεκριμένες τιμές)
   ↓
Συνάρτηση  →  εκτελεί μια ΠΡΑΞΗ
   ↓
Script  →  τα συνδέει όλα μαζί
```
