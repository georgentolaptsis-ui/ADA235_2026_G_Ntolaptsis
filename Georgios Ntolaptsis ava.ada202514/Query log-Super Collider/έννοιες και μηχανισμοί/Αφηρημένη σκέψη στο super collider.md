### Σύνδεση με SuperCollider

Στο SuperCollider η αφηρημένη γνώση εμφανίζεται παντού:

**Envelope = αφηρημένη ιδέα του χρόνου**

supercollider

```supercollider
Env.perc(0.01, 1.0)
// δεν λέει "χτύπα τύμπανο"
// λέει "κάτι ξεκινά γρήγορα και σβήνει αργά"
// → αφηρημένη περιγραφή σχήματος
```

**Pattern = αφηρημένη ιδέα της επανάληψης**

supercollider

```supercollider
Pseq([0, 2, 4], inf)
// δεν είναι νότες ακόμα
// είναι η ΙΔΕΑ μιας ακολουθίας
// → γίνεται νότα μόνο όταν συνδεθεί με Pbind
```

**SynthDef = αφηρημένη ιδέα ενός ήχου**

supercollider

```supercollider
SynthDef(\bass, { |freq = 440| ... })
// δεν είναι ήχος ακόμα
// είναι η ΠΕΡΙΓΡΑΦΗ του ήχου
// → γίνεται ήχος μόνο με Synth(\bass)
```

---

### Η Ιεραρχία

```
ΑΦΗΡΗΜΕΝΟ
    SynthDef  →  περιγραφή
    Pattern   →  ιδέα ακολουθίας
    Env       →  σχήμα χρόνου
         ↓
ΣΥΓΚΕΚΡΙΜΕΝΟ
    Synth     →  πραγματικός ήχος που παίζει τώρα
    EventStreamPlayer → συγκεκριμένη εκτέλεση
    EnvGen    →  η περιβάλλουσα που τρέχει αυτή τη στιγμή
```

---

### Γιατί είναι χρήσιμο αυτό

Επειδή στο SuperCollider **χωρίζεις** την ιδέα από την εκτέλεση. Αυτό σου δίνει δύναμη:

supercollider

```supercollider
// Ορίζεις ΜΙΑ φορά αφηρημένα
SynthDef(\drone, { |freq, amp| ... }).add;

// Εκτελείς ΠΟΛΛΕΣ φορές συγκεκριμένα
Synth(\drone, [freq: 100, amp: 0.3]);
Synth(\drone, [freq: 200, amp: 0.2]);
Synth(\drone, [freq: 300, amp: 0.1]);
```
